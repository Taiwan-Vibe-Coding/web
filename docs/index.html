BC Extension

API / MCP

TW BizConnect API

ERP 中介服務層（L2）。提供統一 REST 介面，將正航、鼎新等 ERP 資料接入 AI 工具；ERP 更換時上層 MCP 不需改寫。

HR Portal

TW HR Self-Service

員工自助入口 SDK。請假申請、加班申請、薪資單查詢，透過 BizConnect API 與後端 ERP 同步。

產品類型說明
Taiwan Payroll for BC BC Extension 薪資試算、假勤、勞健保級距、G/L 過帳
Taiwan E-Invoice for BC BC Extension 電子發票管線、ISV Client、上傳 Log
